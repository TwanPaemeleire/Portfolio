2D Game Engine + DigDug

This project was made as part of my Game Development study programme at Howest - Digital Arts and Entertainment over the course of one semester. The goal of this project was to learn about gameplay programming patterns and how/when to use them. As such, we were tasked with making a game engine, and then additionally making an old retro game that was built on it.

Features

Component-driven architecture
Event system
Scenegraph functionality(parent and children objects)
Singleplayer and co-op game-modes
Level loading & high score saving using binary files
Applied multiple other patterns, such as: State, service locator, command, dirty flag, etc.

Technologies Used

Visual Studio as IDE
Github for version control
